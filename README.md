# Репозиторий ВКР "Инструменты компьютерного зрения для описания киноязыка голливудских фильмов 1990-х" Елизаветы Копыловой 

Данные о голливудских фильмах 1990-х годов, полученные с помощью цифровых инструментов для анализа видео.

Датасет расположен [по ссылке](https://zenodo.org/records/20434635).

Содержание датасета:

1. **Datasheet.md** &mdash; паспорт данных
2. **db_scheme.png** &mdash; схема базы данных
3. **data_gathering** &mdash; код для сбора данных
4. **analysis** &mdash; код для анализа данных

Данные собраны для анализа кинематографических трендов. В датасет вошли 298 фильмов Голливуда 1990-х. Метаданные для фильмов собраны по сайту imdb.com
Датасет доступен в формате db (sql-база данных) и отдельными csv-таблицами.

Схема базы данных:

![Схема базы данных](db_scheme.png)

Список использованных инструментов:
1. Деление по монтажных склейкам &mdash; библиотека dvt (Distant Viewing Toolkit)
2. Информация о лицах и сложности кадра &mdash; библиотека opencv
3. Данные по яркости кадра &mdash; библиотека PIL (Python Imaging Library)
4. Метаданные фильмов &mdash; imdb API

Вычисленные параметры могут содержать погрешности из-за специфики исходного видео и использованных инструментов.
