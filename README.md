# Hollywood-90s-Data
Данные о голливудских фильмах 1990-х годов, полученные с помощью цифровых инструментов для анализа видео.




Данные собраны для анализа кинематографических трендов. В датасет вошли 298 фильмов Голливуда 1990-х. Метаданные для фильмов собраны по сайту imdb.com

Формат данных: CSV, JSON или Parquet?

Инструменты: 

Вычисленные параметры могут содержать погрешности из-за качества исходного видео.

Типы данных:

Метаданные: Таблица с колонками (Название, Год, Режиссер, Жанр, Бюджет, Сборы в США и т.д.).

Вычисленные параметры: Связка с конкретным фильмом (Яркость кадра — float, Наличие крупного плана — bool/float confidence, Длина сцены — float в секундах, Сложность кадра — float/integer).
